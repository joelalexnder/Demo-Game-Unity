# First Game (proyecto básico)

Descripción

Este repositorio contiene un juego sencillo y educativo creado como proyecto inicial. Incluye scripts en C#, shaders (ShaderLab/HLSL) y recursos básicos. Está pensado como un ejemplo para aprender mecánicas de juego, efectos visuales y estructura de proyecto en Unity.

Tecnologías

- Unity (C#)
- ShaderLab / HLSL
- C / C++ (utilizados en herramientas o plugins auxiliares)

Cómo ejecutar

1. Abre Unity Hub y añade este proyecto (selecciona la carpeta del repositorio).
2. Abre la escena principal y pulsa Play. Si no se indica la versión en el proyecto, prueba con Unity 2019.4 LTS, 2020.x o 2021.x.
3. Para generar una build, usa File > Build Settings y sigue el flujo según la plataforma objetivo.

Controles (ejemplo)

- Movimiento: WASD o flechas
- Saltar: Barra espaciadora
- Interacción: Click izquierdo / E

(Estos controles son un ejemplo; revisa los scripts en Assets para confirmar los mapeos exactos.)
